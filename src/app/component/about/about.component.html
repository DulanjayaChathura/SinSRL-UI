SinSRL: First Ever Semantic Role Labeller for Sinhala

SinSRL is the first ever semantic role labeller for Sinhala language which uses both “Neural Network” and “Annotation Projection” approaches to annotate a given Sinhala Sentence with semantic tags defined in Propbank corpus.

SinSRL consists of two sub tools as SinSRL-Projector and SinSRL-Direct Annotator. While SinSRL-Projector uses Sinhala and English parallel sentences to annotate the Sinhala sentence using annotation projection method, the SinSRL-Direct Annotator uses only the Sinhala sentence and trained ML models to annotate the Sinhala sentence.
